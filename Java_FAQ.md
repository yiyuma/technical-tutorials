Was ist JDK, JRE und JVM?<br>
JVM (Java Virtual Machine) ist verantwortlich für Ausführung von Java ByteCode.<br>
JRE (Java Runtime Environment) ist die Laufzeitumgebung und ist unabhängig von Betriebsystem. Sprich, es gibt für jedes Betriebsystem eine entsprechende Versionn. JRE enthält ein JVM.<br>
JDK (Java Development Kit) ist das Entwicklungstool für Java und enthält JRE.<br>
